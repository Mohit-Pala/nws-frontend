About
This is a project that uses a variety of machine learning models to analyze news articles. The models used are:
  • Sentiment Model
  • Positive/neutral/negative Model
  • Gemini
  • GPT
The project also uses the Google Search API to search for news articles.
The project is built using Angular and Tailwind CSS.
The project is hosted on Firebase and uses Firestore as a databas
